## 摘要

本项目设计并实现了一套基于行空板M10的智能运动腰带系统。该系统采用穿戴式设计方案，将传感器模块集成于腰带之上，实现对人体运动状态、姿态以及跌倒行为的实时监测。系统以行空板M10为核心控制器，通过板载六轴传感器（ICM20689）采集三轴加速度和三轴陀螺仪数据，结合学术文献中的步数检测、姿态识别和跌倒检测算法，实现运动监测功能。项目采用前后端分离架构，前端提供Web控制界面实现远程监控与数据可视化，后端采用Flask框架构建RESTful API服务。

**关键词**：行空板M10；穿戴式设备；步数检测；姿态识别；跌倒检测；MEMS传感器

## 一、项目概述

### 1.1 研究背景

随着人口老龄化趋势加剧和人们对健康关注度的提升，穿戴式智能健康监测设备已成为物联网应用领域的研究热点。腰部作为人体活动的核心部位，是放置运动传感器的理想位置。本项目基于行空板M10开发平台，设计并实现了一套集成步数检测、姿态识别和跌倒检测功能的智能运动腰带系统。

### 1.2 系统功能

系统支持三种工作模式，满足不同场景下的使用需求：

| 模式 | 编号 | 功能描述 |
|------|------|----------|
| 生活模式 | 0 | 姿态监测、久坐提醒、跌倒检测、温湿度监测 |
| 运动模式 | 1 | 步数统计、配速计算、碳排放追踪、运动时长记录 |
| 会议模式 | 2 | 屏幕休眠、静音、保持基础数据采集 |

### 1.3 系统规格

| 指标 | 数值 | 说明 |
|------|------|------|
| 步数检测准确率 | ≥97% | 基于多阈值算法 |
| 姿态识别准确率 | ≥85% | 坐/站区分 |
| 跌倒检测准确率 | ≥90% | 基于多特征融合 |
| 系统响应时间 | <100ms | 本地处理 |
| 数据上报间隔 | 2秒 | 服务器通信 |

### 1.4 配速计算功能

系统支持两种配速计算方式：GNSS速度计算和步数计算。

#### 1.4.1 GNSS配速计算

当GNSS模块有效时（卫星数量大于4颗），系统每10秒读取一次GNSS速度，根据速度计算配速。计算公式为：

```
配速(分钟/公里) = 60 / 速度(公里/小时)
```

有效值判断条件：30秒内行驶距离大于等于0.0625公里（对应配速8'00"/公里），且配速在3-30分钟/公里范围内。

#### 1.4.2 步数配速计算

当GNSS模块无效时，系统使用步数计算配速。计算逻辑如下：

- 若总步数为0，不计算配速，显示默认值
- 第一次出现步数增加时，开始30秒计时
- 30秒计时结束时，计算步频和距离：根据步频查询对应步长，计算30秒内的行驶距离
- 有效值判断条件与GNSS配速相同
- 若30秒内步数为0，停止实时计算，显示最近有效配速；下次出现步数增加时重新开始计时

#### 1.4.3 无效值处理

配速无效时，显示最近一次有效配速值。若运动期间未出现过有效配速，则显示默认值"--'--\""。

#### 1.4.4 平均配速计算

运动会话结束时，系统计算整个运动期间的平均配速。计算方法为：统计所有有效配速时间段内的总距离和总时间，用总距离除以总时间得到平均配速。

```
平均配速 = 总时间(分钟) / 总距离(公里)
```

#### 1.4.5 GNSS卫星检测

GNSS模块有效性判定：

- 非运动模式（除会议模式外）：每20秒检查一次卫星数量，卫星数量大于4颗时判定为有效
- 进入运动模式时：启动30秒搜星线程，每秒检查一次卫星数量，达到4颗以上时结束搜星
- 搜星超时（30秒内未达到4颗卫星）：播报语音提示"您当前可能处于室内，卫星信号弱，卫星定位暂不可用"，并切换至步数配速计算模式
- 运动期间GNSS持续有效时，系统时间显示优先使用GNSS获取的时间

## 三、核心算法

本项目的核心算法基于学术文献实现，主要涉及步数检测、姿态识别和跌倒检测三个领域。

### 3.1 步数检测算法

#### 3.1.1 算法来源

参考文献：

> 蒋博, 付乐乐. 基于腰部MEMS加速度计的多阈值步数检测算法[J]. 传感器与微系统, 2018.

#### 3.1.2 重力加速度去除

由于近地面地球引力的影响，加速度传感器采集到的数据包含重力加速度分量。只有去除重力后的线性加速度与人体的运动相关。

本系统采用低通滤波器方法去除重力加速度：

```
Ag(n) = α × Ad(n) + (1-α) × Ag(n-1)
Al(n) = Ad(n) - Ag(n)
```

其中：
- Ad: 传感器测量加速度
- Ag: 重力加速度估计值
- Al: 线性加速度（运动产生）
- α: 滤波系数（默认0.3）

参考文献：

> 胡成全, 王凯, 何丽莉等. 基于MEMS六轴传感器的上肢运动识别系统[J]. 吉林大学学报(信息科学版), 2018.

#### 3.1.3 多阈值步数检测算法

采用三阶段检测方法：

**步骤1（初始化）**：
- 窗口大小：7
- 初始步数：Step = 0
- 初始标志位：Flag = 1

**步骤2（波峰检测）**：
- 条件1：C[0] > T_max 且 C[1] > T_max 且 ... 且 C[6] > T_max
- 条件2：C[3] > C[0] 且 C[3] > C[1] 且 ... 且 C[3] > C[6]
- 同时满足：Flag = 2

**步骤3（过零点检测）**：
- 条件1：C[3] < 0
- 条件2：C[2] > 0
- 同时满足：Flag = 3

**步骤4（波谷检测）**：
- 条件1：C[0] < T_min 且 C[1] < T_min 且 ... 且 C[6] < T_min
- 条件2：C[3] < C[0] 且 C[3] < C[1] 且 ... 且 C[3] < C[6]
- 同时满足：Flag = 1，Step = Step + 1

#### 3.1.4 核心参数

| 参数名称 | 默认值 | 说明 |
|---------|--------|------|
| t_max | 0.12g | 波峰阈值 |
| t_min | -0.06g | 波谷阈值 |
| window_size | 7 | 滑动窗口大小 |
| min_interval_ms | 250 | 最小步间隔 |
| max_interval_ms | 2000 | 最大步间隔 |

### 3.2 姿态识别算法

#### 3.2.1 算法来源

参考文献：

> 卢先领, 王洪斌, 王莹莹, 徐保国. 一种基于加速度传感器的人体跌倒识别方法[J]. 计算机应用研究, 2013, 30(4).

> 李毅. 基于加速度传感器的人体姿态识别研究[J]. 民间科技, 2022.

#### 3.2.2 姿态角计算

利用三轴加速度计算俯仰角（pitch）和翻滚角（roll）：

```
俯仰角: pitch = atan2(ay, √(ax² + az²)) × 180/π
翻滚角: roll  = atan2(ax, √(ay² + az²)) × 180/π
```

参考文献：

> 张庆啟. 基于穿戴式设备的人体姿态识别研究[D]. 北京邮电大学, 2018.

#### 3.2.3 姿态判定规则

根据翻滚角范围判断姿态：

| 姿态 | 翻滚角范围 |
|------|-----------|
| 站立 | -90° ~ -70° |
| 坐姿 | -70° ~ -30° |
| 躺卧 | -30° ~ +30° |

---

### 3.3 跌倒检测算法

#### 3.3.1 算法来源

参考文献：

> 李争. 一种基于三轴加速度的跌倒检测方法[J]. 电子科技, 2015, 28(3).

#### 3.3.2 特征提取

根据文献，提取以下四个特征：

| 特征 | 计算公式 | 阈值 |
|-----|---------|------|
| 合加速度 | RA = √(ax² + ay² + az²) | > 1.85g |
| 加速度变化率 | SA = (1/t) × ∫A(t)dt | > 2.2 |
| 加速度能量 | E = ∫s(t)²dt | > 3.5 |
| 方向变化 | D = √[Σ(xi - yi)²] | > 0.3 |

#### 3.3.3 跌倒判定

```
if (RA > 1.85 && SA > 2.2 && E > 3.5 && Dip > 0.3):
    return 跌倒
else:
    return 非跌倒
```
